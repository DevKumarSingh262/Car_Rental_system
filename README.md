<h1>Car Rental System</h1>

<p>A car rental system built in Java. This application allows users to manage car rentals, including adding new cars, renting out cars, and returning cars. It also provides an interface for administrators to manage </p>
the car inventory and view rental history.

<H1>Features</H1>
<li>Car Management: Add, update, and remove cars from the inventory.</li>
<li>User Management: Register new users and manage their rental history.</li>
<li>Rental Transactions: Rent out cars to users and process returns.</li>
<li>Search and Filter: Search for available cars and filter by various attributes.</li>
<li>Reporting: Generate reports on rental transactions and car availability.</li>
