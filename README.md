<h2>Car Rental System</h2>

<p>This Car Rental System is a Java-based application designed to manage car rentals efficiently. The system allows users to perform various operations such as renting a car, returning a car, and viewing available cars. The application leverages core Java libraries to provide a robust and user-friendly experience.</p>


<H2>Features</H2>
<li>View Available Cars: List all cars available for rent.</li>
<li>Rent a Car: Rent a car by providing user details and car choice.</li>
<li>Return a Car: Return a rented car and update its availability.</li>


<h2>Usage</h2>
<ol>
<li>Start the Application: Run the main class to start the application.</li>
<li>Interact with the Menu: Follow the on-screen prompts to perform actions such as renting or returning a car.</li>
<li>View Results: After performing actions, results will be displayed in the console.</li>
</ol>

<h2>Code Structure</h2>
<li>CarRentalSystem.java: The main class containing the application logic and user interface.</li>
<li>Car.java: A class representing the car objects with attributes like model, make, and availability.</li>
<li>RentalRecord.java: A class representing the rental records, including user information and rental details.</li>
